# Calm To-Do List Web App

A minimalist and calming to-do list web application built with Python Flask backend, HTML frontend, and CSS styling.

## Features

- ✅ **Add Tasks**: Simple form to add new tasks
- ✅ **Mark as Done**: Toggle task completion status
- ✅ **Delete Tasks**: Remove tasks you no longer need
- ✅ **Clear Completed**: Remove all completed tasks at once
- ✅ **Statistics**: Track total, completed, and pending tasks
- ✅ **Responsive Design**: Works on desktop and mobile
- ✅ **Persistent Storage**: Tasks saved to JSON file
- ✅ **Calming UI**: Soothing colors and smooth animations

## Tech Stack

- **Backend**: Python with Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: JSON file (no database required)
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)
